# Security Policy

## Supported Versions

The following versions of the application are currently being supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | ✅ Yes             |
| < 1.0   | ❌ No              |

## Reporting a Vulnerability

We take the security of this project seriously. If you believe you have found a security vulnerability, please report it to us by following these steps:

1. **Do not** open a public issue.
2. Send an email to [INSERT SECURITY CONTACT EMAIL] with a detailed description of the vulnerability.
3. We will acknowledge your report within 48 hours and provide an estimated timeline for a fix.

Thank you for helping keep our users safe!
